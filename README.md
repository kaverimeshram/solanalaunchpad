
# Solana Launchpad

A React-based Solana launchpad prototype. This repository contains a frontend app built with Create React App that uses Solana Web3 and SPL Token libraries to demonstrate blockchain interaction.

## Project overview

- App name: `solana-launchpad`
- Built with React 19 and Create React App
- Uses `@solana/web3.js` and `@solana/spl-token`
- Main UI component lives in `src/components/SolanaLaunchpad.jsx`

## Purpose

This project is intended to showcase a Solana launchpad-like experience with wallet integration, token operations, and a React UI.

## Installation

```bash
npm install
```

## Running locally

```bash
npm start
```

Open `http://localhost:3000` in your browser.

## Build for production

```bash
npm run build
```

This generates an optimized production bundle in the `build` folder.

## Scripts

- `npm start` — run the local development server
- `npm test` — run tests
- `npm run build` — create a production build
- `npm run eject` — eject Create React App configuration (one-way operation)
